# IMDB Movie Review Sentiment Analysis using RNN

Problem Statement->
 Classify movie reviews as positive or negative based on textual data using a deep learning model.

Dataset->
- IMDB Movie Review Dataset
- Contains thousands of labeled movie reviews
- Binary classification problem (Positive / Negative)
- Text data requiring preprocessing and sequence modeling

Approach->
- Performed text preprocessing including tokenization and padding
- Converted text data into numerical sequences
- Built a Recurrent Neural Network (RNN) for sentiment classification
- Trained the model on labeled review data

Results->
- Achieved Accuracy: 85.5%
- Model effectively captured sequential patterns in text data
- Demonstrated strong performance on sentiment classification task

Model Insights->
- RNN successfully learned contextual relationships in text
- Showed importance of sequence modeling in NLP tasks
- Improved understanding of deep learning for text data

Tools & Libraries->
- Python
- NumPy, Pandas
- TensorFlow
- Natural Language Processing techniques
